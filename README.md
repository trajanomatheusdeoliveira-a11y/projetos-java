#  Portfólio de Projetos: Competências em Desenvolvimento de Sistemas (1º Ano IFSul)

Este repositório contém uma coleção de projetos que representam a **consolidação das competências técnicas** adquiridas durante o primeiro ano do curso **Técnico em Desenvolvimento de Sistemas** no Instituto Federal Sul-Rio-Grandense (**IFSul**).

Os projetos foram estruturados em fases, demonstrando uma evolução progressiva no domínio da lógica de programação e das estruturas fundamentais da linguagem Java.

---

## 1. Fase I: Fundamentos e Estruturas de Controle

Nesta fase inicial, os projetos (`calculadora_temperaturas` e `jogo_adivinhação`) tiveram como objetivo principal estabelecer uma base sólida nos conceitos essenciais de programação. O foco esteve no **domínio da interação básica** e do **controle de fluxo**. 

As competências principais desenvolvidas foram:
* Correta manipulação de **entrada de dados** (classe `Scanner`).
* Aplicação sólida de **estruturas condicionais** (`if-else`).
* Introdução aos **laços de repetição** (`while`) para iteratividade básica.

## 2. Fase II: Estruturas de Dados e Lógica de Negócio

A fase intermediária, representada pelos projetos (`biblioteca` e `mercadinho`), marcou a transição para o uso de **estruturas de dados dinâmicas** e o desenvolvimento de **lógicas de negócio** mais complexas. 

O conhecimento em condicionais e *loops* foi aprofundado e utilizado em conjunto com o domínio de **coleções de dados**, como **Listas** (`List`) e a implementação `ArrayList`, essenciais para a modelagem e gestão de conjuntos de informações.

## 3. Fase III: Síntese e Introdução à Orientação a Objetos (POO)

Os projetos finais (`quizz` e `adventure_text`) funcionam como o ápice do conhecimento fundamental, exigindo a **integração e o uso avançado** de todos os conteúdos trabalhados.

Estes projetos demonstram proficiência em:
* Uso abrangente de **estruturas de controle** (`if-else`, `switch-case`, `while`, `for`, `do-while`).
* Gestão de dados eficaz com **Listas** (`List`, `ArrayList`).
* Aplicação dos **conceitos iniciais de Programação Orientada a Objetos (POO)** em Java.

> **Observação:** O projeto **"adventure\_text"** foi desenvolvido de forma **autônoma** para atuar como um **laboratório de teste**, excedendo os requisitos curriculares. Ele evidencia a **iniciativa e a proatividade** na busca por desafios e na consolidação do conhecimento adquirido no curso.
